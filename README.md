Loja Camisetas de time

 Cadastrar Camiseta
 Listar Camisetas
 Consultar Camiseta por Código
 Excluir Camiseta
 Alterar Camiseta (somente o modelo)
 Buscar Camisetas por Preço Máximo
 Buscar Camisetas por Modelo
 Comparar Camisetas em Promoção
